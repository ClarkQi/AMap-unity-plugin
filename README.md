# AMap-unity-plugin

高德地图、CoreLocation集成

### Required
> 本案例使用的是V6.3.0版本，最新资源请到[官方下载](http://lbs.amap.com/api/ios-sdk/download/)。
- AMapFoundationKit.framework
- AMapLocationKit.framework
- MAMapKit.framework(AMap.bundle)
- AMapSearchKit.framework

### Features
- [x] 实时定位；
- [x] 显示地图；
- [x] 兴趣点；
- [ ] 路径规划；
- [ ] 实时导航；
